What is the Document Object Model (DOM)?

The DOM (Document Object Model) is a structured representation of a webpage, turning the HTML content into a tree of nodes that can be accessed and manipulated using JavaScript.

How do you select an element from the DOM?

You can select an element using JavaScript methods like document.getElementById(),
document.querySelector(), or document.querySelectorAll().
