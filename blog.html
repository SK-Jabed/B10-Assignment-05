What is the Document Object Model (DOM)?

DOM stands for Document Object Model. The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each part of the document (like elements, attributes, and text) is an object that can be accessed and modified using languages like JavaScript.

How do you select an element from the DOM?

To select an element from the DOM, I can use methods like:

  • 1.getElementById()
  • 2.getElementsByClassName()
  • 3.getElementsByTagName()
  • 4.querySelector()
  • 5.querySelectorAll()
These methods allow to target specific elements by their ID, class, tag name or CSS selectors for further manipulation or styling.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element instead of multiple listeners to individual child elements. This parent element can manage all events that bubble up from its child elements. This approach is useful as it reduces the number of event listeners, improves performance and makes it easier to manage dynamic content.

How do you manipulate an element's attributes and styles using the DOM?

There are many ways to manipulate an element's attributes and styles using the DOM:


Manipulating Attributes:

  • 1. element.setAttribute("attribute", "value"): Adds or changes the specified attribute.
  • 2. element.getAttribute("attribute"): Retrieves the value of the specified attribute.
  • 3. element.removeAttribute("attribute"): Removes the specified attribute.

Manipulating Styles:

  • 1. element.classList.add("class-name"): Adds a class.
  • 2. element.classList.remove("class-name"): Removes a class.
  • 3. element.classList.toggle("class-name"): Toggles a class on and off
